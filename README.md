## 📌 Features

* عرض أسئلة متعددة
* اختيار الإجابات بسهولة
* حساب النتيجة النهائية
* واجهة بسيطة وسريعة

---

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3

---

## ▶️ Getting Started

لتشغيل المشروع على جهازك:

```bash
npm install
npm start
```

هيفتح معاك على:
http://localhost:3000

---

## 📦 Build

لإنشاء نسخة جاهزة للنشر:

```bash
npm run build
```
